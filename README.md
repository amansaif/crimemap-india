# 🗺️ CrimeMap India

🚀 **CrimeMap India** is an interactive data visualization web application that provides insights into crime trends across major Indian cities and states.
It helps users understand **safety levels, crime distribution, and trends** using modern UI and real-time visualizations.

---

## 🌟 Features

✨ **City Safety Dashboard**

* View safety scores for 29 major cities
* Interactive map with clickable markers
* Crime trends (2020–2023) visualization

👩 **Women Safety Analysis**

* State-wise crime data
* Detailed breakdown (rape, assault, kidnapping, etc.)
* Safety ranking of states

📊 **Data Visualization**

* Donut charts for crime types
* Line charts for yearly trends
* Safety score indicators (rings)

🗺️ **Interactive Map (Leaflet)**

* Zoomable & draggable map
* Dynamic markers based on safety score
* Popup insights for each city

🔍 **Search & Filter**

* Search cities/states instantly
* Filter by minimum safety score

📱 **Responsive Design**

* Works on desktop, tablet, and mobile

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Visualization:** Canvas API
* **Maps:** Leaflet.js
* **Data:** NCRB-based dataset (static JSON/JS objects)
